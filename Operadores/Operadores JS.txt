Operadores JS

-Aritimetica-
+ - Adição
- - Subtração
* - Multiplicação
** - Exponencial
/ - Divisão
% - Módulo
++ - Incrementar (+1)
-- - Decrementar (-1)

-Atribuição-
= - atribuição
+= - x+=y  (x=x+y)
*= - x*=y  (x=x-y)
/= - x/=y  (x=x/y)
%= - x%=y  (x=x%y) (Resto da divisão dos números)

-Comparação-
== - Igual a
=== - Mesmo valor e mesmo tipo
!= - Diferente
!== - Valor e tipos diferentes
> - Maior que
< - Menor que
>= - Maior ou igual
<= - Menor ou igual

-Logica-
&& - e
|| - ou
! - não

-Condicional-
? : - funciona como if e else (if tenario)
ex:
(condition) ? expression1 : expression2
  condição      true          false
